# orderbook-poc-task

## install dependencies

```bash
pnpm install
```

## run backend (RUST)

```bash
cd backend-rust 
cargo run
```

## run frontend

```bash
cd frontend
pnpm build && pnpm start
```

## run backend (JS)

```bash
cd backend
pnpm start
```
    

## View FPS

CMD + SHIFT + P -> FPS (Show frames per second)
